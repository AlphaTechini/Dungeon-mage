# Web3 Route

This standalone route demonstrates Magic Email OTP, ZeroDev Kernel account creation, the address to fund, a capped match, a scoped session key, and Arbitrum Sepolia transaction links. It does not import the Excalibur game runtime.

To find the user-facing integration flow visit [+page.svelte](file:///C:/game-hackathons/Expedition%2033-DN/src/routes/web3/+page.svelte).

The Magic, ZeroDev, and Arbitrum connections can be found in [../../lib/web3/README.md](file:///C:/game-hackathons/Expedition%2033-DN/src/lib/web3/README.md).
